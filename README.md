# NV-center-simulation-2
Python simulation of phonon-driven spin dynamics in nitrogen-vacancy centers in diamond.

## Overview

This repository contains my undergraduate research project investigating phonon-driven spin manipulation in nitrogen-vacancy (NV) centers in diamond.

The project models the time-dependent Hamiltonian of the NV center and simulates spin dynamics under magnetic, hyperfine, and strain interactions.
